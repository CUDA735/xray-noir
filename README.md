# X-Ray 1.6: Noir 
This project is a specialized fork of the X-Ray 1.6 engine (S.T.A.L.K.E.R.: Call of Pripyat).
The primary goal of the fork is to provide mod developers with powerful tools for directing cinematic scenes and creating complex stealth mechanics entirely through Lua scripts, minimizing the need to modify the engine core.
The project is built with a focus on code cleanliness, stability, and convenience for developing story‑driven mods.
